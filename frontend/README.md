# Store Rating Application

## Overview
This is a Full Stack Store Rating Application developed using React.js, Node.js, Express.js, and MySQL. It supports three roles: Admin, User, and Store Owner.

## Technologies Used
- React.js
- Node.js
- Express.js
- MySQL
- JWT Authentication
- bcrypt

## Features
- User Signup
- User Login
- Role-based Authentication
- Admin Dashboard
- User Dashboard
- Owner Dashboard
- Add User
- Add Store
- Search Users
- Sort Users
- Submit Rating
- Update Rating
- Change Password
- Logout

## How to Run

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Database

Create a MySQL database and configure the `.env` file with your database credentials.
